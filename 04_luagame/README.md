# Lua文字游戏 - 武侠之路

一个基于Lua的文字对话游戏，通过Node.js服务器在网页中运行。

## 游戏剧情
1. 游戏开始
2. 选择剑客/刀客/拳师
3. 挑战守卫（守卫防御200，攻击200）
4. 挑战失败
5. 学习独孤九剑
6. 再次挑战
7. 挑战成功
8. 走出新手村
9. 游戏结束

## 运行要求
- Node.js 8+
- Lua（可选，未安装时使用JavaScript模拟）

## 安装和运行

### 方法1：快速开始（无需安装Lua）
```bash
# 1. 下载项目
git clone <repository-url>
cd lua-game-project

# 2. 运行服务器
node game_server.js

# 3. 访问 http://localhost:3000
